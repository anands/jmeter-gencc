## JMeter GenCC

JMeter plugin to generate test credit cards numbers.

## Installing:

```sh
# 1. Clone the repo

# 2. Run:
mvn clean package

# 3. Copy the `jmeter-gencc-1.0.0.jar` from target folder to $JMETER_HOME/lib/ext
cp target/jmeter-gencc-1.0.0.jar $JMETER_HOME/lib/exts

# 4. Restart the JMeter
```
